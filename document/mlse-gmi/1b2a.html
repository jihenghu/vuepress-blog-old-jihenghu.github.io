# - 数据介绍

GMI1B数据集是GPM亮温产品，详细信息见GPM官网, 以及 GMI1B Full Documentation (ATBD).

GMI2A数据集采用的是 2A-GPROF-GMI，提供单幅轨道的GPROF降水猜测，GPROF ATBD.

GMI的诸多通道参数如下，重点关注其分辨率大小，这些分辨率都是没有resample的。 ttp:

# -下载

使用Wget进行下载，脚本如下：

getGMI.sh

#!/bin/bash
for yymm in `cat $1`; do
    for dd in `cat $2`; do
        dpath=$yymm/$dd'/1B/*GMI*' 
        ##dpath=$yymm/$dd'/gprof/2A.GPM.GMI.GPROF*'  ###下载GMI2A的路径
        echo 'start wget :'$dpath '****************************************************'
        # wget
        wget -c -r -N --ftp-user=hjh18305@gmail.com --ftp-password=hjh18305@gmail.com ftp://arthurhou.pps.eosdis.nasa.gov/gpmdata/$dpath
    done
done
